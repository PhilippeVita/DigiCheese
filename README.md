# DigiCheese

DigiCheese est une application informatique de Gestion d'une Pâtisserie "DigiCheese", dans le cadre d'un Projet d'étude au sein du Centre DIGINAMIC.

La conception de l'application a entièrement été élaborée à l'aide du langage UML 3.
Et, l'implémentation du Back-end de l'application est développée sous Python 3.11.9.
